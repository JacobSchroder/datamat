# Music Production Setup

These apps and VST plugins are installed manually (not managed by Homebrew) and are excluded from drift detection.

## 1. Ableton Live

- Go to [ableton.com](https://www.ableton.com)
- Log in with your **custom domain email**
- Download and install Ableton Live

## 2. Native Instruments

- Go to [native-instruments.com](https://www.native-instruments.com)
- Log in and download **Native Access**
- Open Native Access and install all apps and extensions from there

## 3. Plugin Boutique (VSTs)

- Go to [pluginboutique.com](https://www.pluginboutique.com/)
- Log in with your **custom domain email**
- Download and install all purchased VSTs

## 4. Xfer Records — Serum

- Go to [xferrecords.com](https://www.xferrecords.com)
- Log in with your **Gmail**
- Download and install **Serum 1** and **Serum 2**

## 5. Cableguys — ShaperBox

- Go to [cableguys.com](https://www.cableguys.com)
- Log in with your **Gmail**
- Download and install **ShaperBox 2** and **ShaperBox 3**
